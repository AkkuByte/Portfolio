# Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, experience, and the various projects I've worked on as a web developer. It is designed to be fully responsive, interactive, and visually appealing, featuring a dark/light mode toggle.

## 📖 About The Project

This portfolio is built to highlight my expertise in web development and serves as a central hub for my professional identity. It includes sections for:
*   **About Me**: A brief introduction to my background and passions.
*   **Skills**: A grid showcasing the languages and technologies I am proficient in.
*   **Projects**: A curated list of my web development projects with descriptions.
*   **Contact**: A working contact form to get in touch with me directly.

## ✨ Features

*   **Responsive Design**: Fluid layout that adapts perfectly to desktop, tablet, and mobile screens.
*   **Theme Toggle**: Users can switch between Dark Mode and Light Mode, with their preference saved in local storage.
*   **Modern UI/UX**: Built with modern typography (Inter and Syne fonts), smooth transitions, and engaging hover effects.
*   **Functional Contact Form**: Integrated with Formspree for handling form submissions without needing a backend.

## 🛠️ Built With

This project was built from scratch using core web technologies:

*   **HTML5**: Semantic markup for structure.
*   **CSS3**: Custom styling, CSS Variables for theming, Flexbox, and CSS Grid for layout.
*   **Vanilla JavaScript**: DOM manipulation for theme toggling and interactivity.

## 🏃‍♂️ Getting Started

To run this project locally, follow these simple steps:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/AkkuByte/Portfolio.git
    ```
2.  **Open the directory**
    ```bash
    cd portfolio
    ```
3.  **Run the project**
    Simply open the `index.html` file in your preferred web browser. Alternatively, you can use a local development server like VS Code's "Live Server" extension for a better development experience.
